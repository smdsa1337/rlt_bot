<h1>Приложение имеет следующие переменные окружения:</h1>

| name        | default value          | description                                         |
|-------------|------------------------|-----------------------------------------------------|
| APP_NAME    | rlt_bot                | Название приложения, используется в логгере         |
| APP_VERSION | -                      | Версия приложения, используется для создания образа |
| LOG_LEVEL   | INFO                   | Уровень логирования                                 |
| BOT_TOKEN   | -                      | Токен бота                                          |
| PG_HOST     | -                      | Хост БД                                             |
| PG_PORT     | -                      | Порт БД                                             |
| PG_USER     | -                      | Пользователь БД                                     |
| PG_PASSWORD | -                      | Пароль пользователя БД                              |
| PG_DBNAME   | -                      | Название базы данных                                |
| LLM_URL     | http://localhost:11434 | Ссылка до локально развернутой llm                  |
| LLL_MODEL   | -                      | Название llm модели                                 |

Загрузка предоставленного json`а происходит автоматически при запуске (если вдруг строка является дубликатом, она будет проигнорирована)

Чтоб запустить локально, нужно всего лишь
```bash
$ make up
```

Зависимости:

* make 
* docker 
* docker-compose
